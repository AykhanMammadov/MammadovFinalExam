Ilia State University GPA Calculation Rule

At Ilia State University, GPA (Grade Point Average) is calculated using the Grade Point received in each course and the number of credits assigned to that course.

GPA = Σ(each course GP × each course credit) / Σ(each course credits)

Grade Point Conversion

Score Range Letter Grade GP Value
0 - 50 F 0.00
51 - 60 E 1.00
61 - 70 D Increases by 1/12 for each point
71 - 80 C Continues increasing by 1/12 for each point
81 - 90 B Continues increasing by 1/12 for each point
91 - 95 A Continues increasing by 1/12 for each point
96 - 100 A+ 4.00

Final GPA is rounded to one decimal place according to the official rule.
